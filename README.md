# 오목 돌 놓기 데모

간단한 오목 보드를 메모리 상에서 관리하며, 플레이어가 입력한 좌표에 돌을 두는 기능을 제공합니다. 이미 돌이 놓인 위치인지, 보드 범위를 벗어났는지를 검사하여 올바른 수만 허용합니다.

## 실행 방법

```bash
python gomoku.py
```

프로그램은 `B`와 `W` 플레이어가 교대로 수를 두도록 안내합니다. 좌표는 `x y` 형태의 두 정수를 입력하며, `q`/`quit`/`exit`를 입력하면 종료됩니다.

## 웹 UI 레이아웃 미리보기

- `index.html` 파일을 브라우저로 열면 헤더, 메인 게임 영역, 점수판/로그 사이드패널이 포함된 기본 페이지 구조를 확인할 수 있습니다.
- 보드 영역은 19x19 격자로 표현되어 있으며 차후 Canvas 또는 DOM 요소로 돌을 렌더링할 수 있도록 비워두었습니다.
- 사이드패널에는 플레이어 점수, 진행 로그, 조작 버튼 등 필수 UI 요소가 자리잡고 있어 추후 기능 구현 시 구조를 그대로 활용할 수 있습니다.
